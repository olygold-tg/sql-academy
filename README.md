3. Вывести все рейсы, совершенные из Москвы

```mysql
SELECT * FROM Trip
WHERE town_from = 'Moscow'
```

4. Вывести имена людей, которые заканчиваются на "man"

```mysql
SELECT name 
FROM passenger 
WHERE name LIKE '%man'
```

5. Вывести количество рейсов, совершенных на TU-134

```mysql
SELECT COUNT(*) AS count
FROM trip
WHERE plane = 'TU-134';
```
